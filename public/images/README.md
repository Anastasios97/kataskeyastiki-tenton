# Φωτογραφίες ιστοσελίδας

Οι φωτογραφίες τοποθετούνται εδώ με τα ακριβή ονόματα που αναφέρονται στο
`PHOTO-GUIDE.md` της ρίζας του project.

Παράδειγμα:

```text
public/images/pergola/hero.jpg
public/images/pergola/gallery-01.jpg
public/images/pergola/gallery-02.jpg
```

Δεν χρειάζεται αλλαγή στον κώδικα. Μόλις υπάρχει το αντίστοιχο αρχείο, η
ιστοσελίδα το χρησιμοποιεί αντί για την προσωρινή εξωτερική φωτογραφία.
